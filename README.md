# AWS Java Reverse Proxy Deployment

This project demonstrates deployment of a Java web application on AWS using Nginx reverse proxy and Apache Tomcat.

## Architecture

User → Nginx → Apache Tomcat → AWS RDS MySQL

## Technologies Used

- AWS EC2
- Nginx Reverse Proxy
- Apache Tomcat
- Java JSP / Servlet
- AWS RDS MySQL

## Features

- Student Registration Form
- Store student data in database
- View Students List
- Edit and Delete Records
